# Sprint 6 — Privacy and Documentation

## Goal

Give users explicit control over every sensitive data category and make documentation match actual behavior.

## Work items

- Add a global pause-all-tracking control.
- Allow app-usage tracking to be disabled independently.
- Add application and website exclusion lists.
- Add location automation enable/disable controls.
- Explain which map, routing, font, and sync services receive network requests.
- Display permission status for Location, Notifications, Apple Events, Camera, and Microphone.
- Provide links to macOS settings when permission is denied.
- Add clear export, retention, deletion, and backup documentation.
- Rename heuristic “AI” features to “Smart” unless an actual model is involved.
- Update the security policy with current supported versions and a real contact address.
- Update installation, upgrade, Homebrew, recovery, and troubleshooting documentation.

## Exit criteria

- Every collected data category can be disabled.
- Users can see when data may leave the Mac.
- Documentation matches the implementation and supported versions.
- Users can view, export, back up, restore, and delete their information.
