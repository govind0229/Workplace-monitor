# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.6.x   | :white_check_mark: |
| < 1.6.0 | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover any security-related issues, please do NOT use the public issue tracker. Instead, please follow these steps:

1. Send an email to [security@example.com](mailto:security@example.com) with details of the vulnerability.
2. Provide a proof of concept (PoC) if possible.
3. Give us a reasonable amount of time to respond and fix the issue before public disclosure.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
